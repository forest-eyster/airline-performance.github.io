# Airline Performance

## Introduction
Hello, welcome to my explorative data analysis on airline performance. 
This analysis focused on exploring root causes of delays and where to address improvements.


### The questions I wanted to answer through my analysis were:
- Which airlines have improved (or worsened) their on-time performance year-over-year from 2013 onward?
- In the last 11 years what type of delays cause the most time for airlines and airports? 
- Which states or cities have the worst aggregate delay experince?
- How do the top 5 utilized airlines compare on every metric head-to-head?
- How does delay rate vary by month/season, and which carriers are most exposed to seasonal swings?
